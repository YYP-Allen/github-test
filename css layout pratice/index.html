球员介绍

  • 史蒂芬.库里

    2009年首轮第七顺位被勇士队选中，新秀赛季入选最佳新秀第一阵容；2014-15、2016-17、2017-18赛季三次随勇士队获得NBA总冠军；两次当选常规赛MVP，两次入选最佳阵容第一阵容，5次入选全明星赛西部首发阵容。

  • 克莱.汤普森

    2011首轮第十一顺位被勇士队选中，新秀赛季入选最佳新秀阵容第一阵容；2014-15、2016-17、2017-18赛季三次随勇士队获得NBA总冠军；2次入选最佳阵容第三阵容，4次入选全明星阵容。

  • 凯文.杜兰特

    2007首轮第二顺位被西雅图超音速队选中。2016年7月，转会加盟金州勇士在2016-17、2017-18赛季两次随勇士队获得NBA总冠军及FMVP，9届NBA全明星阵容；6届NBA最佳阵容一阵。
